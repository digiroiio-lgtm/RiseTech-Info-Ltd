# RiseTech Information Ltd

Premium consulting services landing page for RiseTech Information Ltd — a strategic advisory firm serving founders, investors, and digital businesses.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm run start
```

## Page Sections

| Component | Description |
|---|---|
| `Navbar` | Fixed top navigation with logo and CTA |
| `HeroSection` | Full-screen hero with headline and CTAs |
| `TrustSection` | Credibility positioning with bullet grid |
| `ServicesSection` | Three service categories with cards |
| `ServiceCategory` | Category wrapper with numbered heading |
| `ServiceCard` | Individual service card with price |
| `FeaturedService` | Highlighted Programmatic SEO Plan |
| `WhyUsSection` | Four value proposition blocks |
| `ClientTypesSection` | Five client type tiles |
| `CTASection` | Final call-to-action with contact links |
